Dataset: Global Air Pollution Dataset

Source:
https://www.kaggle.com/datasets/hasibalmuzdadid/global-air-pollution-dataset

File required in this folder:
global_air_pollution_dataset.csv

How to obtain the data:
1. Open the Kaggle dataset page above.
2. Sign in to Kaggle if required.
3. Download the dataset archive.
4. Extract the CSV file named global_air_pollution_dataset.csv.
5. Place global_air_pollution_dataset.csv inside this data folder.

Expected project path:
data/global_air_pollution_dataset.csv

The Streamlit app and analysis code expect this exact relative path.
